## 1. 系统设置

提升输入响应，减少延迟。

**1.1 键盘连发速度**

- 推荐：控制面板 → 键盘（或运行 `control keyboard`）
   - 重复延迟：最短
   - 重复速度：最快
   - 适合网吧，注销后不易丢失
- 进阶：注册表 `HKEY_CURRENT_USER\Control Panel\Keyboard`
   - `KeyboardDelay=0`，`KeyboardSpeed=31`
   - 修改后需重启或重新登录
   - Windows 有上限，调到头即极限

**1.2 关闭鼠标加速**

- 路径：设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标选项 → 指针选项
- 取消勾选“增强指针精确度（Enhance pointer precision）”

## 2. 游戏画质优化

提升画面流畅度与清晰度。

**移除 3D 阴影，保留 blob 阴影（推荐 High 档）**
1. 游戏设置中将画质/贴图调到 High
2. 退出游戏，编辑 `Documents/StarCraft II/variables.txt`，末尾追加：
   ```ini
   GraphicsOptionShadowQuality=0
   shadowmapsize=1
   ```
3. 效果：高画质下无 3D 阴影但保留 blob 阴影
4. 若 blob 阴影消失，重加上述两行，避免在 Options 里改设置

> 💡 高端显卡（RTX 40/50）可直接开极致画质+抗锯齿

## 3. 游戏内设置

优化操作体验，减少干扰。

**3.1 游戏玩法**
- 勾选除“启用简易命令卡”外所有复选框
- “启用选择敌方单位”建议关闭，防止技能误选

**3.2 声音设置**
- 取消勾选“响应声音”“环境声音”

**3.3 颜色设置**

| 对象 | 颜色 |
|------|------|
| 自己 | 🔵 蓝色 |
| 对手 | 🔴 红色 |
| 中立 | ⚪ 白色 |

## 4. 快捷键设置

提升操作效率，减少误触。

**4.1 镜头（Camera）**
| 功能         | 默认键      | 建议改为           | 说明           |
|--------------|-------------|--------------------|----------------|
| 基地镜头     | Backspace   | Space              | 更好按         |
| 创建位置1-4  | -           | Ctrl+F1/F3/F4/F5   | 编屏           |
| 跳到位置1-4  | -           | F1/F2/F3/F4        | 与创建对应     |
> F2 不推荐改掉

**4.2 编组（Control Groups）**
| 功能         | 建议键位    | 说明               |
|--------------|-------------|--------------------|
| 编组10       | W           | 常用来编基地       |
| 添加到编组10 | Shift+W     | -                  |
| 创建编组10   | Alt+W       | -                  |

**4.3 单位管理（Unit Management）**
| 功能         | 建议键位    | 说明               |
|--------------|-------------|--------------------|
| 折跃门       | Caps Lock   | 折跃门相关         |
| 警告信息     | Backspace   | 与基地镜头对调     |
| 空闲工人     | ` 或 ~      | Tab 上方那颗键     |

**4.4 快速施法（Rapid Fire）**
- 给 TargetChoose 增加额外按键，实现按住键+鼠标连续施放
- 示例：技能绑定 C，按住 C 拖动鼠标可快速重复施放
- 修改方法：
   1. 编辑 `.SC2Hotkeys`，找到 `[Hotkeys]` 段
   2. `TargetChoose=LeftMouseButton,Z,X,C,Q,E,R,T,A,S,D,F,G`
> ⚠️ 快速施法键越多，建造建筑时越易误触发，需要适应

## 5. 个人快捷键方案

> 🚧 以下为个人定制方案（见 basic.SC2Hotkeys），仅供参考

**主要思路**
| 序号 | 改动                | 说明                         |
|------|---------------------|------------------------------|
| 1    | F1-F4 编屏          | 快速跳转战场关键位置         |
| 2    | 1-4 编组            | 主力部队编组集中左手         |
| 3    | 折跃门→传送门       | 不设快捷键，避免误操作       |
| 4    | 传送门→折跃门       | W，选择所有折跃门也是 W      |
| 5    | Q → 巡逻            | 替代默认 P，更好按           |
| 6    | G → H（Hold）       | 空出 G 键用于其他用途        |
| 7    | 不朽者建造          | Q（原为 E）                  |
| 8    | 圣母舰建造          | Q                            |
| 9    | 狂热者冲锋/航母技能 | 不设快捷键，减少误触         |
| 10   | 自爆球技能位置      | 改成和单位面板一致           |
| 11   | 哨兵幻象            | 仅保留执政官、凤凰幻象       |

**已知问题与调优方向**
- 快速施法冲突：TargetChoose 绑定过多按键，建造 BG 易误触
- CV 编组：原计划用 C/V 编组星空之门和 VR，冲突多未实现
- 空白热键方案：empty.SC2Hotkeys 可避免冲突但操作成本高，已放弃
- 当前策略：标准布局渐进优化，配合键盘打字练习
