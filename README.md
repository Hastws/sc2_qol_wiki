## 1. 系统设置

### 1.1 键盘连发速度优化（推荐）

#### 方法一：控制面板设置（推荐网吧使用）

1. 打开 **控制面板 → 键盘**（或运行 `control keyboard`）
2. 调整以下选项：
   - **重复延迟（Repeat delay）**：调到 **最短**
   - **重复速度（Repeat rate）**：调到 **最快**
3. 点击"应用"

> 📝 网吧环境推荐此方法，注册表修改在注销后可能被还原。

#### 方法二：注册表调整（可选，更精确）

运行 `regedit` 打开注册表编辑器。

路径：`HKEY_CURRENT_USER\Control Panel\Keyboard`

| 键名 | 值 | 说明 |
|------|-----|------|
| `KeyboardDelay` | `0` | 最短延迟 |
| `KeyboardSpeed` | `31` | 最快速度 |

> ⚠️ 修改后需要**重新登录或重启**才能生效  
> 📝 Windows 的键盘重复速度有系统上限，调到头就是极限了

### 1.2 关闭鼠标加速

**路径**：设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标选项 → 指针选项

❌ 取消勾选 **"增强指针精确度（Enhance pointer precision）"**

## 2. 游戏画质优化

### 移除 3D 阴影，保留 blob 阴影（推荐 High 画质）

#### 步骤

1. **设置游戏画质**：在游戏设置中将 **Graphic Settings**（及 Textures）调到你想要的档位（如 High）
2. **退出游戏**：关闭 StarCraft II
3. **修改配置文件**：在 `Documents/StarCraft II/` 目录下找到 `variables.txt`，在末尾追加：

```ini
GraphicsOptionShadowQuality=0
shadowmapsize=1
```

**效果**：在 Medium/High/Ultra/Extreme 画质下移除 3D 阴影，同时保留 blob 阴影（游戏内选项无法实现）

#### ⚠️ 注意

如果 blob 阴影突然消失：
- **原因**：在 Options 里更改任何设置（甚至点 OK）都可能导致 blob 阴影消失
- **解决**：重新在 `variables.txt` 末尾追加上述两行，尽量避免在 Options 里修改设置

> 💡 **高端显卡用户**（如 RTX 40/50 系列）可以直接开最高画质，同时开启抗锯齿（Anti-Aliasing），效果很好。

## 3. 游戏内设置

### 3.1 游戏玩法

✅ 勾选**除"启用简易命令卡"以外**的所有复选框

> ⚠️ **关于"启用选择敌方单位"**：建议关闭。例如凤凰释放技能时如果同时存在快捷施法，可能误选敌方农民或单位，影响操作连贯性。而且攻防不影响基本决策，没必要查看。

### 3.2 声音设置

❌ 取消勾选：
- **响应声音**
- **环境声音**

### 3.3 颜色设置

| 对象 | 颜色 |
|------|------|
| 自己 | 🔵 蓝色 |
| 对手 | 🔴 红色 |
| 中立 | ⚪ 白色 |

## 4. 快捷键设置

### 4.1 全局 - 镜头（Camera）

| 功能 | 默认键 | 建议改为 | 说明 |
|------|--------|----------|------|
| 基地镜头 | `Backspace` | `Space` | 更好按 |
| 创建位置 1-4 | - | `Ctrl+F1/F3/F4/F5` | 编屏 |
| 跳到位置 1-4 | - | `F1/F2/F3/F4` | 与创建对应 |

> 📝 F2 不推荐改掉

### 4.2 全局 - 编组（Control Groups）

| 功能 | 建议键位 | 说明 |
|------|----------|------|
| 编组 10 | `W` | 多出一个好按的编组位，常用来编基地 |
| 添加到编组 10 | `Shift+W` | - |
| 创建编组 10 | `Alt+W` | - |

### 4.3 全局 - 单位管理（Unit Management）

| 功能 | 建议键位 | 说明 |
|------|----------|------|
| 折跃（Warpgate） | `Caps Lock` | 折跃门相关 |
| 切换到最近的警告信息 | `Backspace` | 与基地镜头对调 |
| 空闲工人 | `` ` `` 或 `~` | Tab 上方那颗键 |

### 4.4 快速施法（Rapid Fire）

**原理**：给"选择技能或 AI 目标（TargetChoose）"增加额外按键，实现按住键 + 鼠标连续施放

**示例**：把某技能绑定到 C 后，按住 C 拖动鼠标，单位会沿鼠标路径快速重复施放（如火蟑螂种地雷）

#### 手动修改热键文件

1. 找到热键配置文件：`文档/StarCraft II/Accounts/<账号ID>/<子ID>/Hotkeys/`
2. 用记事本打开你的 `.SC2Hotkeys` 文件，找到 `[Hotkeys]` 段落
3. 添加或修改 `TargetChoose` 这一行，在 `LeftMouseButton` 后用逗号分隔追加需要的字母键：

```ini
TargetChoose=LeftMouseButton,Z,X,C,Q,E,R,T,A,S,D,F,G
```

> ⚠️ 快速施法绑定的键越多，在建造建筑（如 BG）时越容易误触发，需要适应。

## 5. 个人快捷键方案

> 🚧 以下是个人定制方案（对应 `basic.SC2Hotkeys`），仅供参考

### 基本思路

| 序号 | 改动 | 说明 |
|------|------|------|
| 1 | 编屏幕用 `F1-F4` 组合 | 快速跳转战场关键位置 |
| 2 | 编组用 `1-4` 组合 | 主力部队编组集中在左手区域 |
| 3 | 折跃门→传送门 | 不设快捷键（避免误操作） |
| 4 | 传送门→折跃门 | `W`；选择所有折跃门也是 `W` |
| 5 | `Q` → 巡逻（Patrol） | 替代默认 P，更好按 |
| 6 | `G` → `H`（Hold） | 空出 G 键用于其他用途 |
| 7 | 不朽者建造 | `Q`（原为 `E`） |
| 8 | 圣母舰建造 | `Q` |
| 9 | 狂热者冲锋/航母技能 | 不设快捷键（减少误触） |
| 10 | 自爆球技能位置 | 改成和单位面板一致的位置 |
| 11 | 哨兵幻象 | 仅保留执政官、凤凰幻象，关闭其他以减少误操作 |

### 已知问题与调优方向

- **快速施法冲突**：大量按键绑定到 `TargetChoose` 后，建造 BG 等操作容易误触，需要适应肌肉记忆
- **CV 编组**：原计划用 `C`/`V` 编组星空之门和 VR（双击跳转），但冲突较多，暂未实现
- **空白热键方案**：曾尝试基于 `empty.SC2Hotkeys` 从零搭建，可以完全避免键位冲突和误触，但操作成本过高（需要手动绑定所有常用功能），最终放弃
- **当前策略**：基于标准布局渐进修改，配合键盘打字练习提升手速
