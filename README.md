## 1. 系统设置

### 1.1 键盘连发速度优化（推荐）

#### 方法一：控制面板设置

1. 打开 **控制面板 → 键盘**（或运行 `control keyboard`）
2. 调整以下选项：
   - **重复延迟（Repeat delay）**：调到 **最短**
   - **重复速度（Repeat rate）**：调到 **最快**
3. 点击"应用"

#### 方法二：注册表调整（可选，更精确）
打开注册表的方法 regedit

路径：`HKEY_CURRENT_USER\Control Panel\Keyboard`

| 键名 | 值 | 说明 |
|------|-----|------|
| `KeyboardDelay` | `0` | 最短延迟 |
| `KeyboardSpeed` | `31` | 最快速度 |

> ⚠️ 修改后需要**重新登录或重启**才能生效  
> 📝 Windows 的键盘重复速度有系统上限，调到头就是极限了
这种方法在网吧可能需要注销，一旦注销，配置就可能还原，所以比较麻烦，可以使用控制面板来调整。

### 1.2 关闭鼠标加速

**路径**：设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标选项 → 指针选项

❌ 取消勾选 **"增强指针精确度（Enhance pointer precision）"**

## 2. 游戏画质优化

### 移除 3D 阴影，保留 blob 阴影（推荐 High 画质）

#### 步骤

1. **设置游戏画质**：在游戏设置中将 **Graphic Settings**（及 Textures）调到你想要的档位（如 High）
2. **退出游戏**：关闭 StarCraft II
3. **修改配置文件**：在 `Documents/StarCraft II/` 目录下找到 `variables.txt`，在末尾追加：

```ini
GraphicsOptionShadowQuality=0
shadowmapsize=1
```

**效果**：在 Medium/High/Ultra/Extreme 画质下移除 3D 阴影，同时保留 blob 阴影（游戏内选项无法实现）

#### ⚠️ 注意

如果 blob 阴影突然消失：
- **原因**：在 Options 里更改任何设置（甚至点 OK）都可能导致 blob 阴影消失
- **解决**：重新在 `variables.txt` 末尾追加上述两行，尽量避免在 Options 里修改设置

现在的 40 系列显卡以及更好的 50 系列显卡，可以开最高的配置，同时，可以开启边界平滑，效果也还不错。

## 3. 游戏内设置

### 3.1 游戏玩法

✅ 勾选**除"启用简易命令卡"以外**的所有复选框

> ⚠️ **关于"启用选择敌方单位"**：建议关闭。例如凤凰释放技能时如果同时存在快捷施法，可能误选敌方农民或单位，影响操作连贯性。而且攻防不影响基本决策，没必要查看。

### 3.2 声音设置

❌ 取消勾选：
- **响应声音**
- **环境声音**

### 3.3 颜色设置

| 对象 | 颜色 |
|------|------|
| 自己 | 🔵 蓝色 |
| 对手 | 🔴 红色 |
| 中立 | ⚪ 白色 |

## 4. 快捷键设置

### 4.1 全局 - 镜头（Camera）

| 功能 | 默认键 | 建议改为 | 说明 |
|------|--------|----------|------|
| 基地镜头 | `Backspace` | `Space` | 更好按 |
| 创建位置 1-4 | - | `Ctrl+F1/F3/F4/F5` | 编屏 |
| 跳到位置 1-4 | - | `F1/F2/F3/F4` | 与创建对应 |

> 📝 F2 不推荐改掉

### 4.2 全局 - 编组（Control Groups）

| 功能 | 建议键位 | 说明 |
|------|----------|------|
| 编组 10 | `W` | 多出一个好按的编组位，常用来编基地 |
| 添加到编组 10 | `Shift+W` | - |
| 创建编组 10 | `Alt+W` | - |

### 4.3 全局 - 单位管理（Unit Management）

| 功能 | 建议键位 | 说明 |
|------|----------|------|
| 折跃（Warpgate） | `Caps Lock` | 折跃门相关 |
| 切换到最近的警告信息 | `Backspace` | 与基地镜头对调 |
| 空闲工人 | `` ` `` 或 `~` | Tab 上方那颗键 |

### 4.4 快速施法（Rapid Fire）

**原理**：给"选择技能或 AI 目标（TargetChoose）"增加额外按键，实现按住键 + 滑鼠标连续施放

**示例**：把某技能绑定到 C 后，按住 C 拖动鼠标，单位会沿鼠标路径快速重复施放（如火蟑螂）

#### 手动修改热键文件

1. 找到热键配置文件：`文档/StarCraft II/.../Hotkeys/`
2. 用记事本打开，找到 `TargetChoose=LeftMouseButton` 这一行
3. 在已有按键后面用逗号分隔添加字母（最后一个不加逗号）

```
TargetChoose=LeftMouseButton,Z,X,C,Q,E,R,T,A,S,D,F,G
```
